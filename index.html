Plot Types

8 Specialized Plot Canvases

Each plot type is purpose-built for a specific geoscience or engineering workflow, with tailored axis handling, data formatting, and export structure.

1D Well Log

Vertical depth track for painting single-curve well log data. Points auto-connect into a continuous depth-sorted trace. Configurable curve name, depth range, and value axis.
Sketch GR, RHOB, NPHI, DT curves · Digitize raster logs · Generate test data

2D Cross Plot

Standard X vs Y scatter plot for cross-plotting any two parameters. Fully configurable axes with custom labels. Perfect for property relationships.
Porosity vs Permeability · Any X-Y relationship · Presentation data

Histogram

Frequency distribution painter. Click to build bars up, right-click to decrease. Configurable bins, value range, and parameter name with count labels.
GR sand/shale cutoffs · Porosity thresholds · Distribution illustration

Multi-Track Well Log

Three side-by-side tracks (GR + RHOB + NPHI) on a shared depth axis. Paint each track independently. Merged 4-column CSV and full LAS 2.0 export.
Composite log view · Multi-curve test data · Training materials

Ternary Plot

Triangular diagram for three-component compositional data. 10% gridlines, barycentric coordinates, and customizable vertex labels (Sand-Silt-Clay, etc.).
Lithology classification · Mineral composition · Fluid fractions

Fence / Decline Curve

Piecewise linear plot with auto-connected control points and shaded area beneath. Sorted by X for clean decline profiles and type curves.
Production decline curves · Rate-time profiles · Pressure transients

Semi-Log / Log-Log

Cross plot with logarithmic scaling. Three modes: Semi-Log X, Semi-Log Y, and full Log-Log. Proper decade gridlines with major and minor ticks.
Pickett plots · Permeability transforms · Multi-decade data

Zone / Fill

Dual-curve depth track with two paintable boundaries (Left/Right). Automatic shaded fill between curves and a configurable cutoff line for threshold visualization.
Net pay intervals · Lithology zones · Neutron-density crossover
Export Formats

Your Data, Your Way

Every plot supports multiple export formats so your painted data goes wherever you need it -- spreadsheets, industry software, or slide decks.

.CSV

CSV Export

Download or copy to clipboard. Column structure is tailored to each plot type. Multi-track exports as merged Depth, GR, RHOB, NPHI columns with -999.25 null values. Available on all 8 plot types.

.LAS

LAS 2.0 Export

Industry-standard Log ASCII Standard format with ~VERSION, ~WELL, ~CURVE, ~PARAMETER, and ~A sections. Proper curve mnemonics and units (GAPI, G/CC, V/V). Loads directly into Techlog, Petrel, IP, and Geolog. Available on 1D and Multi-Track.

.PNG

PNG Image Export

One-click screenshot of the current canvas with all gridlines, labels, traces, and styling. Ready for presentations, reports, and publications. Available on all 8 plot types.

Clipboard

Copy CSV data directly to your clipboard. Paste into Excel, Google Sheets, or any text editor. No file download needed -- instant transfer. Available on all 8 plot types.

Features

Built for Scientists and Engineers

Every detail is designed around real geoscience and petroleum engineering workflows.

Click & Freehand Drawing

Place individual points with precision in Click mode, or drag to paint smooth continuous curves in Freehand mode. Adjustable point size (1-12px) and freehand spacing. Toggle with keyboard shortcuts C and D.

10 Color Palette

Distinguish lithologies, zones, or data populations with 10 vibrant color options. Each point remembers its assigned color for multi-population visualizations.

Undo & Point Removal

Undo the last point with Ctrl+Z, double-click any point to remove it, or clear all data to start fresh. Full control over your painted dataset.

Live Data Preview

A scrollable data table in the sidebar shows your painted data in real time. Status bar displays coordinates under the cursor for precise placement.

Zero Dependencies & Offline

One self-contained HTML file. No npm, no CDN, no server. Works completely offline in any modern browser. Just open the file and go.

Industry-Standard LAS 2.0

Well log and multi-track data exports to CWLS LAS 2.0 format with proper headers, curve mnemonics (DEPT, GR, RHOB, NPHI), and units. Ready for Techlog, Petrel, IP, and Geolog.
